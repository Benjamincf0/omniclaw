Tip: How to use a ZIP format?   

What is a ZIP format?

The ZIP is a file format that allows to temporarily compress multiple files into a single file to allow for a faster transfer.


How to extract files compressed in ZIP format?
  1. Press with your right mouse button on the file to decompress
  2. Access the Extract all category (Extraire)
  3. Follow the instructions of the file extraction utility
If you do not have a file extraction utility yet you can download
The trial version of Stuffit on the following site https://my.smithmicro.com/mac/stuffit.html.


How to compress one or multiple files in ZIP format?
  1. Press with the right mouse button on the file(s) to compress (to create a group of multiple files, hold the CTRL key and press with left mouse button on each desired file)
  2. Access the category Send to (Envoyer vers), then Compressed Folder (Dossier compressÃ©)
  3. Your file will then be compressed in ZIP format and sent where your original file is located

If you do not have a file extraction utility yet you can download
The trial version of Stuffit on the following site https://my.smithmicro.com/mac/stuffit.html.
